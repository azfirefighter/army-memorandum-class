# Army Memorandum LaTeX Class

Provides a basic AR 25-50 memo in latex.

Requires XeLaTeX for font handling.

Install Arial font in debian by running 

```bash
sudo apt-get install ttf-mscorefonts-installer
sudo fc-cache
fc-match Arial
```
